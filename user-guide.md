# User Guide

Dokumentasi ini menjelaskan cara menggunakan website The Story Shoot mulai dari login hingga melakukan booking studio photo.

---

# 1. Membuka Website

User dapat membuka website melalui browser pada desktop maupun smartphone.

Halaman utama website akan menampilkan:

- landing page,
- gallery studio,
- kategori photo,
- dan informasi layanan photobooth.

---

# 2. Register Account

Sebelum melakukan booking, user harus membuat akun terlebih dahulu.

Langkah:

1. Klik tombol **Register**
2. Isi data akun:
   - nama
   - email
   - password
   - konfirmasi password
3. Klik tombol **Submit**

Hasil:

- akun berhasil dibuat
- user dapat login ke website

---

# 3. Login

Setelah memiliki akun, user dapat login ke website.

Langkah:

1. Klik tombol **Login**
2. Masukkan email dan password
3. Klik tombol **Masuk**

Hasil:

- user berhasil masuk ke dashboard website

---

# 4. Melihat Studio

User dapat melihat daftar studio photo yang tersedia.

ketegori studio:

- photobooth studio
- self photo studio
- photo studio

Informasi detail studio:

- nama studio
- harga studio
- kategori studio
- gallery studio
- deskripsi studio

---

# 5. Memilih Jadwal Booking

User dapat memilih:

- tanggal booking,
- jam booking,
- dan studio yang tersedia.

Sistem akan otomatis:

- menampilkan slot yang tersedia,
- dan mencegah jadwal booking yang bentrok.

---

# 6. Melakukan Booking

Langkah booking:

1. Pilih kategori studio
2. Pilih tema studio
3. Pilih tanggal
4. Pilih jam booking
5. Klik tombol booking

Hasil:

- data booking berhasil dibuat
- booking tersimpan di database

---

# 7. Upload Bukti Pembayaran

Setelah booking berhasil dibuat, user dapat mengupload bukti pembayaran.

Langkah:

1. Buka detail booking
2. Klik upload pembayaran
3. Pilih gambar bukti transfer
4. Submit pembayaran

Hasil:

- gambar berhasil diupload
- pembayaran tersimpan
- status booking diperbarui

---

# 8. Melihat Riwayat Booking

User dapat melihat seluruh riwayat booking yang pernah dilakukan.

Informasi yang ditampilkan:

- kode booking
- studio
- tanggal booking
- status booking
- status pembayaran

---

# 9. Melihat Detail Booking

User dapat melihat detail booking yang sudah dibuat.

Informasi detail:

- studio yang dipilih
- tanggal booking
- jam booking
- total pembayaran
- status booking
- bukti pembayaran

---

# 10. Cancel Booking

User dapat membatalkan booking apabila belum di bayar.

Langkah:

1. Buka detail booking
2. Klik tombol cancel booking

Hasil:

- booking dibatalkan
- status berubah menjadi dibatalkan
- slot booking kembali tersedia

---

# User Flow

```text
Open Website
      ↓
Register / Login
      ↓
Melihat Studio
      ↓
Pilih Jadwal Booking
      ↓
Booking Studio
      ↓
Upload Pembayaran
      ↓
Lihat Riwayat Booking
```

---

# Conclusion

Website The Story Shoot membantu user melakukan booking studio photo secara online dengan proses yang lebih mudah, cepat, dan praktis dibandingkan sistem booking manual.
